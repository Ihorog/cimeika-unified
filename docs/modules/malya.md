# Malya — Ідеї

## 1. Призначення

**Malya** (Маля) — модуль ідей, творчості та інновацій. Збереження, розвиток та організація ідей від концепції до реалізації.

### Ключові функції:
- Збереження ідей та концепцій
- Категоризація за типом
- Управління статусом та пріоритетом
- Теги та зв'язки
- Відстеження прогресу
- Інтеграція з іншими модулями

## 2. UI-секції

### Frontend компоненти:
- **MalyaView** (`/malya`) — головний екран ідей
- **IdeaCard** — картка ідеї
- Список ідей
- Детальний перегляд
- Редактор ідей

### Основні UI елементи:
1. **Дошка ідей** — всі активні ідеї
2. **Канбан** — організація за статусом
3. **Детальний перегляд** — повна інформація про ідею
4. **Редактор** — створення/редагування ідей
5. **Фільтри** — за категорією, статусом, пріоритетом

## 3. API

### Endpoints:

#### `GET /api/malya/ideas`
Отримати всі ідеї
```json
{
  "ideas": [...],
  "count": 30
}
```

#### `POST /api/malya/ideas`
Створити нову ідею
```json
Request:
{
  "title": "Назва ідеї",
  "description": "Детальний опис...",
  "category": "innovation",
  "priority": "high",
  "tags": ["технології"]
}

Response: MalyaIdea object
```

#### `GET /api/malya/ideas/{id}`
Отримати ідею за ID

#### `PUT /api/malya/ideas/{id}`
Оновити ідею

#### `DELETE /api/malya/ideas/{id}`
Видалити ідею

## 4. Стани

### Можливі стани модуля:
- **idle** — очікування
- **loading** — завантаження ідей
- **ready** — ідеї завантажені
- **error** — помилка завантаження

### Категорії ідей:
- **innovation** — інновація
- **creative** — творча ідея
- **improvement** — покращення
- **experiment** — експеримент

### Статуси ідей:
- **draft** — чернетка
- **active** — активна
- **completed** — завершена
- **archived** — архівована

### Пріоритети:
- **low** — низький
- **medium** — середній
- **high** — високий

## 5. Зв'язки з модулями

### Інтеграція з іншими модулями:
- **PoDija** — планування реалізації ідей
- **Calendar** — часові рамки реалізації
- **Kazkar** — історія розвитку ідеї
- **Ci** — централізоване управління ідеями
- **Nastrij** — емоційний контекст створення ідеї
