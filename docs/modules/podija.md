# PoDija — Події

## 1. Призначення

**PoDija** (ПоДія) — модуль подій, майбутнього та сценаріїв. Таймлайн життя з можливістю планування майбутнього, документування теперішнього та рефлексії над минулим.

### Ключові функції:
- Таймлайн подій (минуле, теперішнє, майбутнє)
- Життєві віхи та досягнення
- Сценарії майбутнього
- Зв'язки між подіями
- Різні категорії (подія, віха, сценарій)
- Статуси виконання

## 2. UI-секції

### Frontend компоненти:
- **PodijaView** (`/podija`) — головний екран таймлайну
- **TimelineNodeCard** — картка події на таймлайні
- Візуальний таймлайн
- Список подій
- Редактор подій

### Основні UI елементи:
1. **Таймлайн** — візуальне представлення подій у часі
2. **Фільтри часу** — минуле/теперішнє/майбутнє
3. **Детальний перегляд** — повна інформація про подію
4. **Редактор** — створення/редагування подій
5. **Фільтри** — за категорією, статусом, тегами

## 3. API

### Endpoints:

#### `GET /api/podija/timeline`
Отримати таймлайн подій
```json
{
  "timeline": [...],
  "count": 50
}
```

#### `POST /api/podija/timeline`
Створити нову подію на таймлайні
```json
Request:
{
  "title": "Назва події",
  "description": "Детальний опис...",
  "timestamp": "2025-01-01T00:00:00Z",
  "type": "future",
  "category": "milestone",
  "tags": ["кар'єра"]
}

Response: TimelineNode object
```

#### `GET /api/podija/timeline/{id}`
Отримати подію за ID

#### `PUT /api/podija/timeline/{id}`
Оновити подію

## 4. Стани

### Можливі стани модуля:
- **idle** — очікування
- **loading** — завантаження таймлайну
- **ready** — таймлайн завантажений
- **error** — помилка завантаження

### Типи часу:
- **past** — минуле
- **present** — теперішнє
- **future** — майбутнє

### Категорії подій:
- **event** — звичайна подія
- **milestone** — життєва віха
- **scenario** — сценарій майбутнього

### Статуси:
- **planned** — заплановано
- **in_progress** — у процесі
- **completed** — завершено
- **cancelled** — скасовано

## 5. Зв'язки з модулями

### Інтеграція з іншими модулями:
- **Calendar** — синхронізація майбутніх подій
- **Kazkar** — історичні події та спогади
- **Ci** — централізована координація подій
- **Nastrij** — емоційний контекст подій
- **Malya** — реалізація ідей як події
- **Gallery** — візуальна документація подій
