# Gallery — Галерея

## 1. Призначення

**Gallery** (Галерея) — модуль для збереження та організації медіа-контенту. Фотографії, відео, аудіо та інші медіа-файли в єдиному організованому просторі.

### Ключові функції:
- Збереження фото та відео
- Організація медіа-контенту
- Альбоми та колекції
- Теги та категорії
- Пошук по медіа
- Інтеграція з іншими модулями

## 2. UI-секції

### Frontend компоненти:
- **GalleryView** (`/gallery`) — головний екран галереї
- **GalleryItemCard** — картка медіа-елементу
- Сітка галереї
- Перегляд елементів
- Завантаження файлів

### Основні UI елементи:
1. **Сітка медіа** — огляд всіх елементів
2. **Детальний перегляд** — повноекранний перегляд
3. **Форма завантаження** — додавання нових елементів
4. **Фільтри** — за типом, тегами, датою

## 3. API

### Endpoints:

#### `GET /api/gallery/items`
Отримати всі медіа-елементи
```json
{
  "items": [...],
  "count": 20
}
```

#### `POST /api/gallery/items`
Створити новий медіа-елемент
```json
Request:
{
  "title": "Назва",
  "description": "Опис",
  "type": "image",
  "url": "https://...",
  "tags": ["природа", "літо"]
}

Response: GalleryItem object
```

#### `GET /api/gallery/items/{id}`
Отримати елемент за ID

#### `DELETE /api/gallery/items/{id}`
Видалити елемент

## 4. Стани

### Можливі стани:
- **idle** — очікування
- **loading** — завантаження медіа
- **ready** — медіа завантажені
- **error** — помилка завантаження

### Типи медіа:
- **image** — зображення
- **video** — відео
- **audio** — аудіо
- **document** — документ

## 5. Зв'язки з модулями

### Інтеграція з іншими модулями:
- **Kazkar** — медіа-вкладення до історій
- **PoDija** — візуалізація подій
- **Calendar** — прив'язка медіа до подій
- **Ci** — централізований доступ до медіа
